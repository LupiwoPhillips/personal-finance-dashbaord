# Personal Finance Dashboard

## Overview

Personal Finance Dashboard is a web application that helps users track income, expenses, budgets, and savings goals through an intuitive and responsive dashboard.

The application provides financial insights through visual charts, spending analytics, and budgeting tools that help users make informed financial decisions.

---

## Features

- Dashboard Overview
- Income Tracking
- Expense Tracking
- Budget Management
- Savings Goals
- Spending Analytics
- Responsive Design
- Interactive Charts

---

## Technologies Used

- Vue.js
- JavaScript
- Tailwind CSS
- Chart.js
- Supabase (if applicable)
